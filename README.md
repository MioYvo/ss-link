# ss-link

### [ss-link]("www.ss-link.com")是一个shadowsocks账号提供商，拥有众多节点，本工具用来帮助用户找出ping值最低的节点

#### 依赖环境
* Python 2.7.10
	* requests
	* tqdm
	* lxml
	* pyping

* 安装

	```
	pip install -r requirement.txt
	```

#### 使用方法


```
sudo python get_server.py -su

Or

python get_server.py
```

* to see more help

	```
	python get_server.py -h 
	```
